# TownGuardian Backend


TownGuardian is a backend application developed with **Django** and **Django REST Framework**, designed to manage issues reported by citizens to the local administration.

The platform enables users to report, validate, and manage city issues through a role-based access system (Citizen, Agent, Validator, Admin, and SuperAdmin). Its core features include JWT authentication, user management, city zone management, agent assignment, and issue lifecycle tracking.

The backend exposes a REST API documented with Swagger/OpenAPI and uses PostgreSQL for data storage. The project follows a modular architecture based on Django best practices, making it scalable, maintainable, and easy to extend.
